## How to Play

```
Score:
 O: 0     X: 0
 
   0 1 2 3 4
  ┏━━━━━━━━━┓
0 ┃█┃█┃ ┃ ┃█┃   
1 ┃█┃█┃X┃█┃█┃
2 ┃█┃█┃█┃█┃█┃
3 ┗━━━━━━━━━┛
```


## Features
- Keeps track of player score
- alternates start player
- can ctrl+c out of most inputs nicely
